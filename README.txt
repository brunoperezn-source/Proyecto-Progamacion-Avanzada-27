Sistema de Gestión de Programa de Voluntariado

Este proyecto corresponde al Sistema de Información para la gestión de programas de voluntariado. El sistema permite registrar voluntarios, organizaciones y proyectos, así como asignar voluntarios a proyectos según sus habilidades y disponibilidad. Además, se considera un modo de emergencia por catástrofe, donde se pueden asignar voluntarios de forma rápida a proyectos críticos.

El desarrollo está orientado al cumplimiento de los requisitos solicitados en la asignatura, incluyendo encapsulamiento, colecciones anidadas, uso de mapas, sobrecarga de métodos y un menú de consola para la interacción.

Funcionalidades Principales:
-Crear voluntarios
-Añadir voluntarios
-Añadir organizaciones y proyectos

Requisitos del Sistema:
IDE de Java: Este proyecto fue hecho en NetBeans 21, pero se puede usar Eclipse.
Java Development Kit (JDK).

Instalación y Ejecución:

Descomprime este zip en tu carpeta de preferencia.
Abre Apache NetBeans.
Menu->File Open Project.
Selecciona la carpeta donde instalaste el programa.
Debería de salir todo el programa.
Source Packages-> default packages 
Click derecho a Main.Java
Run file.

1. Agregar voluntarios (desde Excel)
2. Agregar voluntario manualmente
3. Asignar voluntarios a proyecto
4. Eliminar voluntario (por RUT)
5. Mostrar organizaciones
6. Asignación de emergencia
7. Mostrar voluntarios cargados
8. Cargar organizaciones
0. Salir

Si quieres agregar un voluntario desde el Excel de la carpeta inicial, selecciona la primera opción y escríbelo en el formato "nombre.xlsx".
Si quieres agregar un voluntario manualmente usa la segunda opción y sigue las instrucciones.
Para asignar voluntarios primero debes de usar la octava opción, y escribir el Excel utilizado para las organizaciones y proyectos en el formato "nombre.xlsx".
Asignación de emergencia aun no esta programada.
